## Clonar el proyecto

```bash
$ git clone https://github.com/GerardoEgR/reportHub.git
```

## Instalar dependencias

```bash
$ pnpm install
```

## Compilar y correr el proyecto

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```
